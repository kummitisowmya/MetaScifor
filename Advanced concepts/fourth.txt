1:Python is an interpreted, object-oriented,
2:high-level programming language with dynamic semantics.
3:It's high-level built in data structures,
4:combined with dynamic typing and dynamic binding,
5:make it very attractive for Rapid Application Development,
6:as well as for use as a scripting or glue language to connect existing components together.
7:Python's simple, easy to learn syntax emphasizes readability and therefore reduces the cost of program maintenance.
8:Python supports modules and packages, which encourages program modularity and code reuse.
9:The Python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.
